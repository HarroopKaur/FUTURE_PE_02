# AI-Powered UGC Ad System – Laneige

## Project Overview
This project demonstrates a structured prompt engineering system for generating high-converting UGC-style ad content.

## Product
Laneige Lip Sleeping Mask

## Problem
Many users struggle with dry, flaky lips and use ineffective lip products.

## Solution
A prompt-driven system that generates:
- Hooks
- UGC scripts
- CTAs
- Captions

## Prompt Logic
The system follows Direct Response marketing:

Hook → Problem → Solution → Value → Social Proof → CTA

## Tools Used
- Google Gemini (Primary LLM)
- GitHub (Documentation)

## Key Features
- Authentic UGC tone
- Multiple hooks
- Conversion-focused content
- Reusable prompt system

## Real-World Use
This system can be used by:
- Marketing agencies
- D2C brands
- Freelancers
