# Prompt Framework

## Role Assignment:
You are an expert UGC Ad Strategist for D2C beauty brands.

## Product Details:
- Product: Laneige Lip Sleeping Mask
- Key Ingredients: Berry Fruit Complex, Vitamin C, Coconut Oil, Shea Butter
- Primary Benefit: Eliminates flakiness and provides 8-hour overnight hydration
- Core Problem: Dry, flaky lips
- Sensory Appeal: Buttery texture, glossy finish, sweet berry scent
- Tone: Relatable, authentic, best-friend recommendation

## Constraints:
- Script length: 15–30 seconds (50–80 words)
- Use Direct Response structure:
  Hook → Problem → Solution → Value Proposition → Social Proof → CTA
- Avoid salesy language
- Include visual directions

## Output Requirements
Generate:
1. Scroll-stopping hooks
2. Full UGC script
3. CTA variations
4. Platform-ready caption

## Model Used:
Google Gemini

## Why Gemini:
Gemini was chosen because it performs well in generating creative, short-form, conversational content suitable for UGC ads.

---

## Reusability

This prompt framework can be reused for:

- Skincare products
- Fitness brands
- Local businesses
- SaaS tools

Only the following inputs need to change:
- Product name
- Problem statement
- Target audience
- Tone

This makes the system scalable for real-world marketing use.
